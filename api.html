四.评论接口

1.发表评论

  • POST：http://music.lazyer.net/music/comment/save

  • 参数 ：

    1. userId:用户id
    2. comment:内容
    3. module:MATERIAL(模块名称)//详见 数据字典-模块
    4. foreignKey:materialId
    5. replyUserId:被回复用户id

2.评论列表

  • POST：http://music.lazyer.net/music/comment/list

  • 参数

    1. module:MATERIAL(模块名称)
    2. foreignKey:materialId
    3. pageStart:分页
    4. pageSize:分页

3.删除评论

  • POST:http://music.lazyer.net/music/comment/delete

  • 参数

    1. userId
    2. commentId:评论id

十. 其他接口

1.意见反馈

  • URL：http://music.lazyer.net/music/suggest/submit

  • method：POST

  • 参数

数据字典

素材状态

  1. DRAFT(0,"草稿"),//草稿
  2. REVIEW(1,"审核中"),//上传中
  3. REJECT(2,"未通过"),//上传失败，需要收到重新上传，或者系统定时批量上传
  4. PUBLISHED(3,"已发布"),//已经发布的
  5. UNSHELVE(4,"下架");//下架

模块

  1. MATERIAL,//素材/音乐
  2. ARTICLE,//赏文
  3. MOMENTS,//朋友圈/乐坛

乐谈类型

  1. TEXT, //文字
  2. IMAGE, //图片
  3. VIDEO, //视频
  4. LINK, //链接
